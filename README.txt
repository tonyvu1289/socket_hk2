File báo cáo là report.pdf, nằm trong thư mục document. ~\document\report.pdf
Trong thư mục document có thư mục report chính là thư mục lưu trữ code latex dùng để viết báo cáo và các thành phần liên quan.
File thực thi là socket.py nằm trong thư mục source. ~\source\socket.py
Chạy chương trình: socket.py>Open with>Python. Chương trình sẽ cho ra địa chỉ IP để truy cập trang web.
